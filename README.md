# Bioinformatics Small Projects

A collection of bioinformatics analysis projects implemented in Python using Biopython.

## Projects

### 1. DNA Sequence Analysis
- Calculates GC content
- Counts nucleotide composition
- Performs transcription and translation

### 2. Open Reading Frame (ORF) Finder
- Identifies ORFs in forward and reverse complement strands
- Translates each ORF to protein sequence
- Calculates protein mass

### 3. Comparative Sequence Analysis
- Compares COX1 gene between Homo sapiens and Mus musculus
- Performs global pairwise alignment
- Calculates Hamming distance and similarity percentage

## Tools
- Python 3
- Biopython
- NCBI FASTA sequences
